# Apache Airflow

O Airflow é uma plataforma para orquestrar e monitorar pipelines de dados. Por permitir o desenvolvimento dos fluxos de forma programática (em python, o que o torna bem flexível) e por ter integração com as principais tecnologias de bigdata, por ora foi escolhido como principal ferramenta de orquestração do time.

**Front-end**

| Ambiente   | endpoint                                 |
|:-----------|:-----------------------------------------|
| PRD        | <https://airflow.data.intranet:8080/>    |
| QA         | <https://airflow.qa.data.intranet:8080/> |

Utilize seu login de rede para autenticar (apenas usuários do grupo  g\_s\_p\_bigdata\_X estão autorizados)
