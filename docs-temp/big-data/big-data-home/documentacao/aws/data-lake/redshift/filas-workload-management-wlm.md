# Filas workload management WLM

| Nome da fila    | Prioridade   | Grupo Redshift    | Predicados                                                                                                                                                   | Ação   | Concurrency scaling   | Descrição                                                                                                                                                                                                                                        |
|:----------------|:-------------|:------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------|:----------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **admin**       | High         | g\_bigdata\_admin | Nested loop join row count (rows) > 100                                                                                                                      | abort  | Não                   | Destinada a usuários com perfil de admin do time de Big Data.                                                                                                                                                                                    |
| **ad\_hoc**     | Normal       | g\_ad\_hoc        | Query execution time (seconds) > 180 <br/> Nested loop join row count (rows) > 100 <br/> Return row count (rows) > 10000 <br/> Spectrum scan (MB) > 100000   | abort  | Sim                   | Destinada a usuários com perfil de exploração de dados, seja via Metabase ou outra uma IDE como dbeaver, por exemplo.                                                                                                                            |
| **development** | Normal       | g\_development    | Query execution time (seconds) > 600 <br/> Nested loop join row count (rows) > 100 <br/> Return row count (rows) > 100000 <br/> Spectrum scan (MB) > 100000  | abort  | Não                   | Destinada a usuários com perfil de desenvolvimento no Redshift.                                                                                                                                                                                  |
| **elt**         | High         | g\_elt            | Nested loop join row count (rows) > 100                                                                                                                      | abort  | Não                   | Destinada a usuários de serviço de elt, etl e orquestração (airflow, spark etc).                                                                                                                                                                 |
| **reporting**   | Normal       | g\_reporting      | Query execution time (seconds) > 600 <br/> Nested loop join row count (rows) > 100 <br/> Return row count (rows) > 2000000 <br/> Spectrum scan (MB) > 100000 | abort  | Não                   | Destinada a usuários de serviço de report (Power BI).                                                                                                                                                                                            |
| **default**     | Lowest       | N/A               | Query execution time (seconds) > 1 <br/> Spectrum scan (MB) > 1                                                                                              | abort  | Não                   | Caso o usuário não esteja presente em nenhum grupo do Redshift mapeado neste tabela (coluna Grupo Redshift), a fila default será utilizada na sessão do usuário. <br/> Esta fila tem uma prioridade baixíssima e contém regras mais restritivas. |
