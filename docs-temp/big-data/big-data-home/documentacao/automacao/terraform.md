# TerraForm

O [Terraform](https://www.terraform.io/) permite criar, alterar e melhorar a infraestrutura de forma segura e previsível. É uma ferramenta de software livre e multiprovider que codifica APIs em arquivos de configuração declarativa, os quais podem ser compartilhados entre os membros da equipe e versionados.
