# Hue

O Hue (Hadoop User Experience) agrupa vários projetos do ecossistema Hadoop em uma interface web. Ele atua como um front-end para aplicativos executados no cluster, permitindo a interação com estes através de interface amigável para o usuário. Por exemplo, através do Hue é possível executar queries no Impala/Hive e executar Jobs Spark.
