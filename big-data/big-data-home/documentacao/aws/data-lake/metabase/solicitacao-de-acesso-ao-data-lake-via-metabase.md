# Solicitação de acesso ao Data Lake via Metabase

Já tem usuário criado no Metabase? Se não, é só acessar a ferramenta pela primeira vez: <https://metabase.intranet:3000/>

Após a criação do usuário, é necessário abrir uma [requisição](https://jira.intranet.uol.com.br/jira/servicedesk/customer/portal/48/create/849) no portal de Governança de Dados, na qual deve ser informado o dado que será acessado e o**porquê do acesso**. Após a aprovação, envie um e-mail para l-bigdata solicitando a liberação no metabase com o **nome/login das pessoas autorizadas**, o **nome da equipe** e o **número da requisição de Governança.**
