# Kafka-Connect

![](https://miro.medium.com/max/669/1*b4BKirX_6DVPmiTr8-sg9w.png)

O Kafka Connect é um framework de código aberto desenvolvido para padronizar a integração de outros sistemas de dados com o Apache Kafka, simplificando o desenvolvimento, a implantação e o gerenciamento de conectores. Ele pode ser executado em modo autônomo, ou seja, uma única máquina é responsável pelo processamento (isso pode ser útil para coleta de logs, por exemplo), ou em modo distribuído, o que o torna escalável e tolerante a falhas.
