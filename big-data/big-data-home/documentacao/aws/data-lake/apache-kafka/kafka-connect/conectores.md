# Conectores

O Kafka connect tem dois tipos de conectores:

**Source Connector**

Tem a função de trazer dados de fontes externas para o Kafka. Uma fonte externa pode ser: Banco de dados, sistemas de arquivos como S3, HDFS, etc.

**Sink Connector**

Tem a função de levar os dados de um tópico Kafka para uma fonte externa, que também pode ser um banco de dados, um sistema de arquivos, etc.
