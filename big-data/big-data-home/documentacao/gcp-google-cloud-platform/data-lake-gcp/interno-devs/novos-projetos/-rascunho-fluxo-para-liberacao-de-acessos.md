# **Acesso aos dados**

## Fluxo atual - manual

- Pedidos via canal: [D&A Datalake - Notificações | Acessos | Microsoft Teams](https://teams.microsoft.com/l/channel/19%3A5143f371d4974549b8e8da78bb73d0e0%40thread.tacv2/Acessos?groupId=d9beaec3-d0b6-4b51-8e4f-bf0d0eb8bbdf&tenantId=7575b092-fc5f-4f6c-b7a5-9e9ef7aca80d)
- Acesso a nível de dataset para os grupos de cada datalab

## Novo fluxo via IDM

- Pedidos via IDM: Pesquisa por "datalake - gcp"
- Acessos a nível de Zona do Dataplex (Domínio + Camada)
  - Usuários ganham acesso a todos os datasets da zona
- Grupo de acesso aos dados não ficam mais atrelados ao grupo de acesso ao Datalab
