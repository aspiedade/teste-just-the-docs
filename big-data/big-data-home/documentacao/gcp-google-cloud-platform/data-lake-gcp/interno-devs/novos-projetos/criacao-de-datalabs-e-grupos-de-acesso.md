# **Criação de novos Datalabs e Grupos de acesso**

## **1) Criação do Grupo no AD**

Abrir chamado: <https://jira.intranet.uol.com.br/jira/servicedesk/customer/portal/45/create/4938>

No campo descritivo citar:

- Nome do grupo no formato: **g\_uol\_gcp\_datalake\_[<sigla-dominio>](/confluence/pages/viewpage.action?pageId=443295146)**
- Pedir para incluir email ao grupo, seguindo mesmo padrão de nome: **g\_uol\_gcp\_datalake\_[<sigla-dominio>](/confluence/pages/viewpage.action?pageId=443295146)@[uolinc.com](http://uolinc.com)**
- As pessoas a serem inseridas neste grupo.

## **2) Sincronizar Grupo AD no GCP**

Abrir chamado: <https://jira.intranet.uol.com.br/jira/servicedesk/customer/portal/45/create/4947>

- Sistema: **Azure**
- Tipo de solicitação: **Adição de grupo em aplicativo empresarial**
- **Nome do aplicativo: APP GCP UOLINC**
- No campo descritivo citar:
  - Favor vincular o grupo **g\_uol\_gcp\_datalake\_[<sigla-dominio>](/confluence/pages/viewpage.action?pageId=443295146)** ao **APP GCP UOLINC no Azure**

## **3) Criação do projeto Datalab no GCP**

Abrir chamado: <https://jira.intranet.uol.com.br/jira/servicedesk/customer/portal/33/create/5182>

Sec Projetos + Sistemas base estão criando o novo fluxo pelo Jira e não tem como pedir projetos atualmente.

Centro de custo (Conta de billing):

- **UOLCS Analytics**: 12971 - Operações Dados
- **UOLCS Publicidade**: 11033 - Operações Publicidade
